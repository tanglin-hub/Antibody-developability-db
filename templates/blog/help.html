Search

Text-based search: First, select which dataset to search. Then, enter your antibody name, or property name. If the search is empty, we will return all the antibody information.

Search Results

The search resulte list:

1.Search by name:If you search for antibodies' name, all the physicochemical property information contained in this antibody will appear.

2.Search by property:If you search for physicochemical properties, all the antibodies and the specific data of this property will appear instead of the complete physicochemical property information.

3.Download:On the search list page, we provide all the information about the target search. Click to download all the antibody information of the target search.

Antibody Detail

1.Antibody visualization:A 3Dmol.js-enabled visualization window shows The three-dimensional structure information of antibodies can be displayed through methods such as cartoon below. Action Options: Rotate: Press and hold the left mouse button. Zoom: Use mouse's scroll button.

2.Download:On the search details page, we offer download options for individual antibodies. You can choose to download the sequence of the antibody in fasta format or all the information about this antibody.

3.Basic information:Here, the unique ID of the included antibody in the DevoAb database, AB0001, is displayed. In addition, the literature source of the antibody (click to jump to the corresponding literature), display format, and sequence information are also shown.

4.Peoperties:Presents all the information on the developability properties of the antibody as experimentally determined.
